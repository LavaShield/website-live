ArchLinux:

I've used a lot of Linux distributions, and the only remotely usable one has been ArchLinux. When you get it to work, it's perfect. However, installing it and getting it to that point is annoying and pointless. If you just want something that works, when you've booted the Arch installer just do a 'sudo pacman -Syu archinstall' and then use the archinstall command to get an Arch install. This will leave you with a functioning Arch Installation with all the essentials you need to use your machine.

During install (for archinstall application):

sudo pacman -Sy archinstall

Setup Script:


# Arch Linux TTY and Graphical Session Shortcuts

Ctrl + Alt + F1 - Switch to TTY1 (usually graphical session or text console)

Ctrl + Alt + F(2-6) - Switch to TTY(2-6) (text console)

Ctrl + Alt + F7 - Switch to TTY7 (usually graphical session)

Ctrl + Alt + F(8-12) - Unused virtual console (blank screen) or additional graphical session
